APP DESCRIPTION
So You Think You Can Science is a science trivia app geared towards high school students. 

Topics in the application include Physics, Chemistry, and Biology.

You will have unlimited time to answer each question. However, to get points for the question, the 
question must be answered within the first 30 seconds

The panel at the bottom of the quiz application displays various statistics for the current session.
The Bar graph button on the same panel displays the number of questions answered correctly based on topic. 
Please enlarge the window when you click on the "Show Bar Graph" button to see the graph.
